# WebSocket
HerokuでデプロイするためのWebsocketのリポジトリ。

wss://kss-pc-club-websocket.herokuapp.com/

## 開発方法
1. `git clone https://github.com/kss-pc-club/websocket.git`
2. `npm install`
3. `npm start`
