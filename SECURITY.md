# Security Policy

## 推奨環境

| Application | Version  | Supported          |
| :---------: | :------: | :----------------: |
| Node.js     | > 12     | :white_check_mark: |
| Node.js     | < 11     | :x:                |
| npm         | > 6.13.0 | :white_check_mark: |
| npm         | < 6.12.0 | :x:                |

## 脆弱性の報告

脆弱性を発見した場合には、Issueにてお寄せください。<br>
もし修正していただける場合には、PRしていただけると幸いです。
