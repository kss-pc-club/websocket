# Security Policy

## 推奨環境（以下は例です。このリポジトリに合わせて変更してください）

| Application | Version  | Supported          |
| :---------: | :------: | :----------------: |
| Node.js     | > 12.x.x | :white_check_mark: |
| Node.js     | < 11.x.x | :x:                |
| Yarn        | 1.22.x   | :white_check_mark: |
| Yarn        | < 1.21.x | :x: |

## 脆弱性の報告

脆弱性を発見した場合には、Issueにてお寄せください。<br>
もし修正していただける場合には、PRしていただけると幸いです。
